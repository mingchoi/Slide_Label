Slide Label
==============

A jquery plugin show a beautiful sliding label when mouseover a link

How to use:
-----------

```$("#myLink").Slide_Label( $("<div id='myDiv' style='width: 150px' >content</div>") );```

or

```
$('#myLink').Slide_Label(
  $('<div>', {id: 'myDiv', style: 'width: 150px', text: 'content'})
)
```
